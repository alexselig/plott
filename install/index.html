Install Plott for PowerPoint

Plott adds a task pane to PowerPoint so you can design a chart, drop it onto the current slide as a picture or editable shapes, and restyle a chart that's already there — right where you're presenting.

Step 1 — get the manifest

A small .xml file that tells PowerPoint where to load Plott from. Download it, then follow your platform below.

Download manifest

Step 2 — side-load it

Pick where you use PowerPoint. It takes about a minute.

PowerPoint on the web

office.com · Microsoft 365

  1. 1
    Open any presentation at office.com (or from OneDrive / SharePoint).
  2. 2
    On the ribbon go to Home → Add-ins (or Insert → Add-ins), then More Add-ins.
  3. 3
    Open the My Add-ins tab and choose Upload My Add-in.
  4. 4
    Browse to the plott-manifest.xml you downloaded and click Upload.
  5. 5
    Click the Chart builder button in the Plott group on the Home tab to open the pane.

PowerPoint on Windows

Microsoft 365 · 2208 or newer

  1. 1
    Put the manifest in a folder, e.g. C:\Plott.
  2. 2
    In File Explorer, right-click that folder → Properties → Sharing → Share… and share it with yourself, then copy the network path (looks like \\YOURPC\Plott).
  3. 3
    In PowerPoint go to File → Options → Trust Center → Trust Center Settings → Trusted Add-in Catalogs.
  4. 4
    Paste the network path into Catalog Url, click Add catalog, tick Show in Menu, then OK and restart PowerPoint.
  5. 5
    Home → Add-ins → My Add-ins → Shared Folder, pick Plott, then use the Chart builder button.

PowerPoint on Mac

Microsoft 365 · 16.64 or newer

  1. 1
    Open Finder, press ⇧⌘G, and go to:
    ~/Library/Containers/com.microsoft.Powerpoint/Data/Documents/wef
    If the weffolder doesn't exist, create it.
  2. 2
    Copy plott-manifest.xml into that wef folder.
  3. 3
    Quit and reopen PowerPoint, then open any presentation.
  4. 4
    Home → Add-ins → My Add-ins → Plott, or click the Chart builder button in the Plott group on the Home tab.

What you can do with the pane

  • Design a chart
    Pick a type, type or paste data, choose a palette and style — live preview as you go.
  • Expand & drag to edit
    Open the full-pane editor and drag bars or points to change values, just like the web app.
  • Insert on the slide
    Drop it as a crisp image, or as native, editable PowerPoint shapes you can recolor.
  • Restyle what's there
    Select a Plott chart already on a slide and re-open it to tweak and update in place.
  • Style an Excel chart
    Select a native chart, pull its data into Plott, restyle it, and overlay it exactly.

Good to know

  • Plott runs entirely in your PowerPoint — your slides and data stay on your device and in your Microsoft 365 session. There's no separate account to create.
  • Style Excel Chart needs to read the deck's data. If a presentation carries an encrypting sensitivity label or password, PowerPoint hands the add-in an encrypted copy it can't read — remove/lower the label (or Save As an unlabeled copy) and reopen, then try again. You can still design a chart and insert it either way.
  • Requires PowerPoint on the web, Windows (Microsoft 365 2208+), or Mac (16.64+). The manifest points at this site, so keep it reachable over HTTPS.
Trouble loading it? Make sure you uploaded the manifest you downloaded from this page, that you're signed in to Microsoft 365, and that you restarted PowerPoint after adding it. Source & issues on GitHub.
